Metadata

Every PDF carries hidden information about its creation, modification, and authorship, invisible to the casual reader but embedded in the file itself.

What is Metadata?

Metadata is "data about data", information that describes the document but isn't part of its visible content. Unlike a printed book, where production details might appear on a colophon page, PDF metadata is embedded in the file's code structure.

What Gets Recorded

  • Author , Who created the document
  • Title , The document's name
  • Creator , The software used to create it
  • Producer , The software that generated the PDF
  • Creation Date , When the file was first made
  • Modification Date , When it was last changed
  • Keywords , Searchable tags
  • Subject , Topic description

Why This Matters

Unlike a printed page, a PDF remembers its history. This has implications for privacy, authenticity, and the nature of digital documents. Metadata can reveal:

  • The original author even after editing
  • What software was used (Word, InDesign, etc.)
  • Exact timestamps of creation and edits
  • The operating system it was made on

Print Comparison

A printed book might include a colophon listing the printer, typeface, and paper, but this is chosen information, curated by the publisher. PDF metadata is automatic, often recorded without the author's awareness. The digital document keeps a hidden record that the printed page cannot.
