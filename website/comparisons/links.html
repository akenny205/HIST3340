Hyperlinks

PDFs can contain clickable connections to other documents, websites, and locations within the file itself, creating a web of intertextuality that print can only approximate.

Types of Links in PDFs

Hyperlinks in PDFs go far beyond simple web URLs:

  • External URLs , Links to websites
  • Internal navigation , Jump to other pages
  • Table of contents , Clickable chapter links
  • Footnote references , Instant access to notes
  • Email links , Open mail client
  • File links , Open other documents
  • Embedded actions , Trigger JavaScript

Print's Limitations

Printed texts developed their own systems for cross-referencing:

üìñ Print Cross-References

  • "See page 47"
  • Footnotes at page bottom
  • Endnotes in back matter
  • Bibliography citations
  • Index entries

üîó PDF Hyperlinks

  • Click to jump instantly
  • Hover to preview
  • Open external resources
  • Back button navigation
  • Searchable destinations

Changed Reading Behavior

Hyperlinks fundamentally change how we read. Instead of linear progression through pages, readers can jump between sections, chase references, and explore tangentially. This mirrors how we browse the web, but embedded in what looks like a "page."

The Problem of Link Rot

External hyperlinks are fragile. Websites disappear, URLs change, and servers shut down. A PDF from 2010 may contain dozens of dead links, digital footnotes pointing to nothing. Print citations, while requiring manual lookup, never "break."

How Links Work in PDFs

Link Annotation Object 
<<
  /Type /Annot
  /Subtype /Link
  /Rect [72 650 200 670]
  /Border [0 0 0]
  /A <<
    /Type /Action
    /S /URI
    /URI (https://example.com)
  >>
>>

Defines clickable region and destination

Internal Link (Go To Page) 
<<
  /Type /Annot
  /Subtype /Link
  /Rect [100 500 180 520]
  /A <<
    /Type /Action
    /S /GoTo
    /D [5 0 R /FitH 800]
  >>
>>

Jumps to page 5, position 800

Intertextuality Made Instant

Literary scholars speak of "intertextuality", how texts reference and build upon other texts. Print made this visible through citations and allusions. PDFs make it actionable. A reader can instantly access the referenced source, blurring the boundary between the document and its references. The text becomes a node in a network rather than a self-contained object.
